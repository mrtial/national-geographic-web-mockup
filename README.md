# National Geographic Mockup Website

As a practice of my front-end skills using Angular and back-end server construction with Express to build a functional and scalable website, I try to build a mockup website of Fatih Ocak's "National Geographic Concept Design". (https://www.behance.net/gallery/36074729/National-Geographic-Concept-Design)

# Angular + Express
The modern MVC structure single-page-website is build with Angular and Express. 

# JWT Authentication
JWT (JSON Web Token) Authentication is implemented for user login and sign up function.
